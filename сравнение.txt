================================================================================
ПРОБЛЕМЫ RADIX ВЕРСИИ (v31-v7)
================================================================================

1. Бургер кнопки НЕ ВИДНЫ после 7 итераций
   - Причина: canvas PixiJS перекрывает кнопки
   - Решение от Grok AI: pointer-events: none на canvas
   - Статус: Применено в v7, требует проверки

2. Упрощённый дизайн потерял "характер"
   - Ретро-неон стиль был уникальным
   - RADIX — стандартный минимализм (как у всех)

3. Усложнение CSS
   - CSS переменные вместо прямых значений
   - Больше классов (.menu-btn, .btn-secondary, etc.)

================================================================================
РЕКОМЕНДАЦИЯ
================================================================================

ОТКАТИТЬСЯ к ДО-RADIX версии (ретро-неон стиль) + применить фикс pointer-events

Почему:
1. Ретро стиль был оригинальным
2. Кнопки работали (были видны)
3. Меньше CSS сложности

Что сохранить от RADIX:
- pointer-events: none на canvas (фикс от Grok AI)
- Версия badge (vN) — полезно для отслеживания
