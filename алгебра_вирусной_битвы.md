# Алгебра вирусной битвы

## Описание реализации

Создана консольная симуляция вирусной битвы с сеткой 20x32 (640 клеток), где две фракции (VIRUS_A и VIRUS_B) соревнуются за контроль территории.

## Основные особенности реализации:

### 1. Структура данных:
- `CellState` - перечисление состояний клетки (EMPTY, VIRUS_A, VIRUS_B)
- `VirusParams` - параметры вируса для каждой фракции (12 параметров)
- `Cell` - состояние одной клетки (состояние, здоровье, уровень заражения, параметры владельца)

### 2. Алгебра тика:
Функция `simulate_tick()` выполняет 6 фаз:
1. Распространение/размножение
2. Заражение/столкновения
3. Мутации и ослабление
4. События (хаос)
5. Восстановление
6. Проверка победы

### 3. Формулы вычислений:
- **Размножение**: `reproduction_chance = Reproduction / 12 + Speed / 24`
- **Заражение**: `infect_chance = Contagiousness / 12 * (1 - Stealth_соседа / 24) * Virulence / 12`
- **Боевая эффективность**: `capture_chance = attack_power / (attack_power + defend_power)`
- **Нестабильность**: `instability += (control_percent - 70) / 100 * 2` при контроле >70%

### 4. События хаоса:
- Глобальная вспышка: +20% к Contagiousness всем
- Ослабление: -30% здоровья случайным 10% клеток доминирующей фракции
- Супермутация: +5 к случайному параметру лидера
- Коллапс: сброс 20% клеток в EMPTY при контроле >90%

### 5. Победа:
Фракция побеждает при контроле >=99% территории

## Использование:
```python
# Установка параметров для игроков
simulator.set_player_params('A', {
    'aggression': 3,
    'mutation': 2,
    'speed': 4,
    # ... остальные параметры
})

# Запуск симуляции
winner = simulator.simulate_tick()
```

Симуляция полностью детерминирована (с фиксированным seed) и готова к интеграции с основной игрой.
