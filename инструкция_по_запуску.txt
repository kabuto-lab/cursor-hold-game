ИНСТРУКЦИЯ ПО ЗАПУСКУ ПРОЕКТА

1. ОЧИСТКА КЭША И ПЕРЕЗАПУСК:
   - Очистите кэш браузера (Ctrl+Shift+Delete или Cmd+Shift+Delete)
   - Обновите страницу с принудительной перезагрузкой (Ctrl+F5 или Cmd+Shift+R)
   - Закройте и откройте браузер заново

2. ПЕРЕЗАПУСК СЕРВЕРА РАЗРАБОТКИ:
   - Остановите текущий сервер (если запущен)
   - Выполните команду: npm run start
   - Или отдельно:
     - В одном терминале: cd server && npm run dev
     - В другом терминале: cd client && npm run dev

3. ПРОВЕРКА СОСТОЯНИЯ ПРОЕКТА:
   - В правой панели должны отображаться:
     * Заголовок "Virus Parameters"
     * 12 параметров вируса в виде пробирок (сетка 3×4)
     * Счетчик оставшихся очков (12 всего)
     * Кнопка "READY"
   - Старые элементы (Follower Speed, Cursor Size, Theme) должны отсутствовать

4. ЕСЛИ ПРОБЛЕМА СОХРАНЯЕТСЯ:
   - Удалите папку node_modules в client и server
   - Выполните npm install в корне проекта
   - Перезапустите приложение
