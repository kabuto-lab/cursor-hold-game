# Технический стек проекта TOVCH (Digital Hand Holding Game)

## Основные технологии

### Frontend
- **Язык программирования**: TypeScript
- **Графический движок**: PixiJS v8+
- **Сборщик модулей**: Vite
- **Стили**: CSS-in-JS (встроенные в HTML)
- **Архитектура**: ООП с использованием классов

### Backend
- **Язык программирования**: TypeScript
- **Сервер**: Node.js
- **Multiplayer фреймворк**: Colyseus
- **Схемы данных**: @colyseus/schema
- **HTTP фреймворк**: Express.js
- **CORS**: cors

### Сетевое взаимодействие
- **Протокол**: WebSocket (через Colyseus)
- **Синхронизация**: Сервер-клиент с авторитетным сервером
- **Состояния**: Синхронизация через схемы Colyseus

## Графика и визуализация
- **Рендеринг**: PixiJS (WebGL/Canvas)
- **Фильтры**: pixi-filters (pixelate, noise, bloom)
- **Спрайты**: PIXI.Graphics, PIXI.Sprite
- **Анимации**: requestAnimationFrame, PIXI.Ticker

## Архитектура проекта

### Клиентская часть (client/)
- **Точка входа**: src/main.ts
- **Основной класс**: src/game.ts (Game)
- **Схемы данных**: src/types/schema.ts
- **Визуализация битвы**: src/virus-battle-algebra.ts
- **HTML**: index.html (с встроенными стилями)

### Серверная часть (server/)
- **Точка входа**: src/index.ts
- **Комнаты**: src/rooms/HoldingRoom.ts
- **Схемы данных**: src/rooms/schema.ts

## Основные функции
- **Синхронизация курсоров**: Real-time через Colyseus
- **"Держание за руки"**: Визуальная связь между курсорами
- **Вирусная битва**: Алгоритм распространения вирусов
- **Параметры вируса**: 12 параметров (агрессия, скорость и т.д.)
- **Интерфейс**: Сайдбары, пробирки с параметрами
- **Чат**: Внутриигровой текстовый чат
- **Кастомизация**: Настройка параметров вируса

## Платформы
- **Кроссплатформенность**: Web (браузер)
- **Адаптивность**: Responsive дизайн
- **Устройства**: Desktop, Tablet, Mobile (теоретически)

## Сборка и деплой
- **Менеджер пакетов**: npm/yarn
- **Сборка**: Vite (frontend), tsc (backend)
- **Рабочие области**: npm workspaces
- **Запуск**: npm run start (concurrently)
